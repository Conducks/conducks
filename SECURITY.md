# Security Policy

## Supported Versions

We are currently in active development. Only the latest version is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability, please do NOT open a public issue. Instead, please report it to contact@conducks.com.

We will acknowledge your report within 48 hours and provide a timeline for addressing the issue. Once the issue is resolved, a public disclosure will be made.

We appreciate your help in keeping CONDUCKS secure.
