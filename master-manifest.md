## 🚀 2. The 32-Command Unified CLI
The Conducks CLI is a powerful engine designed for deep architectural discovery and project maintenance.

### 🔍 Discovery & Analysis
- `conducks analyze`: The primary induction engine. Performs parallel topological indexing.
- `conducks query`: Fuzzy and Regex symbol lookup against the Synapse ID registry.
- `conducks status`: Generates a structural health manifest identifying **Hotspots** and **Pillars**.
- `conducks context`: Generates an LLM-optimized technical manifest of the project.
- `conducks blueprint`: Auto-generates `BLUEPRINT.md` from the current graph.

### ⚡ Behavioral Tracing
- `conducks trace`: Dijkstra-weighted pathfinding between two functional symbols.
- `conducks impact`: Bidirectional Blast Radius analysis (Upstream/Downstream).
- `conducks flows`: Visualize high-level execution circuits.
- `conducks resonance`: Score structural similarity across different modules.

### 🛡️ Governance & Integrity
- `conducks audit`: Sentinel integrity check for circularities and "Structural Sins."
- `conducks guard`: Regression guard to block commits exceeding risk thresholds.
- `conducks explain`: 6-signal risk decomposition (Gravity, Entropy, Complexity, Churn, etc.).
- `conducks clean`: Nuclear purge of database locks and zombie analysis handles.

### 🏺 Evolution & Mutation
- `conducks diff`: Chronoscopic analysis of structural drift across pulses.
- `conducks drift`: Track authorship and complexity velocity over time.
- `conducks rename`: Graph-Verified Renaming (GVR) with atomic rollback.
- `conducks prune`: Identification and removal of orphaned exports/dead code.

### 🎮 Visual Dashboard
- `conducks mirror`: Launches the real-time **Conducks Mirror** on port `3333`.
- `conducks visualize`: Generates static Mermaid diagrams of current relationships.
